Инструкция по вводу в эксплуатацию проекта «Искусственный староста»

Существуют две возможные конфигурации сети, в которую можно интегрировать систему распознавания лиц, но первые этапы идентичны для обоих конфигураций, поэтому начнем поэтапно.
1)	Для начала подключаем камеру, к компьютеру, который будет играть роль мозга сети. Камера может быть абсолютно любой, но обязательное условие для нее, это автоматический фокус.
2)	Размещаем камеру в рабочую область над входной дверью в школу, в предбаннике.
3)	Скачиваем последнюю версию прикладного программного обеспечения из репозитория на GitHub по ссылке: https://github.com/Fallenhacker29/Artificial_class_headmate_SCH/releases
4)	Устанавливаем последнюю версию Python с сайта: https://www.python.org/downloads/ Используя расширенную установку проставляем все галочки и перезагружаем систему
5)	Устанавливаем сервер MySQL помешаем на него базу данных в находятся таблицы списков класса в формате номер – буква(на английском) Сами таблицы имеют следующий формат полей 
Id (начиная с 1)	Фамилия Имя 	IsInSchool	IsLate	Timestamp
6)	В скаченном архиве находим папку с нужной редакцией программы (x64 или 32/x86), для того чтобы узнать нужную вводим в пуске «cmd», открываем консоль вводим «systeminfo» ищем пункт Тип системы
7) разархивируем нужную папку(или весь архив)
8)запускаем файл install.exe  от имени администратора
9)	Указываем полный путь установки 
10)	Указываем IP адрес  SQL сервера, если работает локально, тогда 127.0.0.1
11)	Готово
