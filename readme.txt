Пояснение:
create_database.py - инструмент создания базы из фото
trainer.py - инструмент обучения нейросети, создает файл, который надо переместить в /face
main.py - запустить обученную нейронную сеть в работу
finder.py - программа обработчик данных выводимых при помощи main
на данный момент не работает, нужно заполнить индексную таблицу в коде
для постройки графиков использовать fictive_plort.py

инструкция по обучению сети:
1) Запустить create_database программа спросит индекс лица в базе см (исходный код main.py массив faces)
2) Запустить trainer.py
3) Можно запускать main.py
Для установки использовать ветку релиза или готовый релиз
